FakeEngine

FakeEngine is a 3D game engine from scratch! I'm fooling around with C++ and low-level C in the hopes of making an engine, in the hopes of making some fun retro-style games out of it. For the moment, I am developing and testing primarily on amd64 Linux machines. However, with the intention of making it cross-platform and portable eventually, all dependencies of FakeEngine are included in the source code. You just need up-to-date versions of CMake, Make, and a C++ compiler to make it happen. No pun intended.

The New Curator

A school project which I volunteered to collaborate on, The New Curator is an educational first person game which places the player as an intern at an art museum. We had 4 artists working on the game, one programmer (myself), a composer/musician, and a voiceover artist. I did all of the logic, controls, and event programming, including audio triggers, item manipulation and interactions, the game menu, and the progression logic.

ToastEngine

2D game engine and physics simulator. This demo features a custom-coded, from scratch, 2D classical kinematics physics engine capable of simulating the motion of particles with defined mass, radius, and other attributes. Various player control schemes are also demonstrated. Among these is a ragdoll drag and drop algorithm which follows the mouse in a smooth or chaotic pattern, depending on how you click. Behind the scenes, ToastEngine is simulating damped oscillations to achieve this effect.
